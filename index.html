Git aur GitHub kya hai? 🤔

Git ek version control system hai jo developers ko apna code manage karne mein help karta hai. Jab hum koi code likhte hain, toh usme changes aate rehte hain – kabhi kuch naya add karna, kabhi bugs fix karna. Git se hum yeh sab changes track kar sakte hain.

Git kya karta hai?

  • 📸 Code ka har version ya snapshot save karta hai
  • ⏪ Kisi bhi purane version pe wapas jaa sakte ho
  • 🌿 Team ke members alag-alag branches pe kaam kar sakte hain

☁️ GitHub kya hai?

GitHub ek online platform hai jahan hum Git ka use karke apna code host karte hain. Matlab: Git local machine pe code manage karta hai, aur GitHub us code ko cloud mein safe rakhta hai — jisse team members ya duniya ke log usse dekh ya use kar sakein.

⚖️ Git vs GitHub – Antar

  • Git:🛠️ Tool hai version control ke liye (offline kaam karta hai)
  • GitHub: 🌍 Website hai jahan Git repos store kar sakte hain (online access ke liye)

💻 Common Git Commands (Simple Examples)

  • git init – 🔁 New Git repo banana
  • git add . – ➕ Saare files ko staging mein lana
  • git commit -m "message" – 💾 Changes save karna
  • git push – 🚀 Changes ko GitHub pe bhejna
  • git status – 📋 Check karna ki kya changes hua hai

🎁 GitHub ke Fayde

  • 🤝 Team collaboration easy ho jaata hai
  • ☁️ Code backup cloud pe hota hai
  • 🌍 Open-source contribution possible hota hai
  • ⚙️ Issue tracking, pull requests, aur CI/CD tools use kar sakte hain
